# 拼豆工坊 (Pindou Workshop)

一个将图片转换为拼豆图纸的 Web 工具。

## 功能特点

- 图片上传与预览
- 智能像素化处理
- CIEDE2000 精确颜色匹配算法
- MARD 拼豆色库支持
- 图像调整（亮度/对比度/饱和度）
- 可交互画布（缩放/平移）
- 用量清单统计
- 图纸导出 (PNG)
- 深色模式支持

## 技术栈

- Vue 3 + TypeScript + Vite
- Pinia 状态管理
- Element Plus UI 组件库
- Tailwind CSS 样式框架
- Canvas 图像处理
- CIEDE2000 色差算法

## 快速开始

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```

## 使用说明

1. 点击上传区域选择图片，或拖拽图片到上传区
2. 调整网格尺寸（默认 52×52）
3. 选择颜色匹配算法（推荐 CIEDE2000）
4. 调整图像参数（可选）
5. 点击「生成拼豆图」
6. 在画布中缩放和查看
7. 导出图纸或复制用量清单

## 色库说明

当前版本包含 MARD 品牌 45+ 种常用颜色，涵盖：
- 基础色（黑白灰/红橙黄绿青蓝紫棕粉）
- 荧光色
- 珠光色

## 颜色匹配算法

### 快速模式 (欧几里得距离)
- 直接计算 RGB 空间中的直线距离
- 处理速度快，适合快速预览

### 精确模式 (CIEDE2000)
- 将 RGB 转换到 Lab 色彩空间
- 使用 CIEDE2000 标准色差公式
- 更符合人眼对颜色的感知
- 推荐用于最终输出
